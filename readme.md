1. First, create a folder
2. Next, create a file. For ex. index.html
3. Then, create a new repository on github
4. Add the new or updated files on the new repository through terminal (git bash)
    - go to terminal
    - apply git bash option
    - check "git status"
    - then to add all the updated files, use the command (git add .). If not, apply the command (git add /filename/)
    - next commit the files with a message that correspond to the updated files. Use the command (git commit -m "yourmessage")
    - lastly, push the new or updated files to the github portal. Apply the command (git push -u origin main)
5. With these steps, all the new or updated files are now saved on the github that can be accessed online.
6. To save the files on your desktop, just copy the local clone from the site. Open cmd then use the command (git clone /link/). Next, find the directory then apply the command (code .)
